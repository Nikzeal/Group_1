What is it?

HTML5 is the fifth and final version of HTML developed by the World Wide Web Consortium (W3C) and is maintained by the Web Hypertext Application Technology Working Group (WHATWG), which is made up by major browser developers. It was released on 22nd January 2008. Its goals are to allow the language to support the latest developments in multimedia content, as well as making content accessible to both humans and automated programs. This was necessary as HTMl4 was released in 2000.

What did it add?

HTML5 mainly added support for new multimedia content. Tags such as <audio>, <video> and <canvas> were added so you no longer had to rely on CSS & JavaScript to implement those types of content. It has been designed so that older browsers can safely ignore the new HTML5 elements.

Timeline

W3C began working on the new standard in 2004 and in May 2011 invited various organisations to review the new specifications. Whilst HTML5 itself was released in 2008, the W3C did not designate it as the "W3C Recommendation" until October 2014.

Future of HTML

HTML5 was retired on 27th March 2018 for what is now known as the "HTML Living Stanard", which is named to indicate that HTML will never be fully complete and will constantly be updated and adapted to the future.
